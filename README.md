# python OOP challenge

## flask commands

```
  create-homestereo              Creates the home stereo
  create-portable-ghettoblaster  Creates the portable ghettoblaster
  routes                         Show the routes for the app.
  run                            Run a development server.
  shell                          Run a shell in the app context.
 ```

## creating the two objects with flask cli

```
$ flask create-homestereo                                                          
<models.HomeStereo object at 0x7feeb437bd60> created by flask CLI.
$ flask create-portable-ghettoblaster                                               
<models.PortableGhettoblaster object at 0x7fe40e37bd60> created by flask CLI.  

```
